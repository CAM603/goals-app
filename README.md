## How it works

1. Create a goal
2. Add steps you need to complete in order to achieve your goal
3. Start crossing off steps, one at a time
4. Complete your goal!

## Features

-   Dark mode
-   SQLite database
    -   Goals and steps will persist

## To Run:

-   Install Expo https://expo.io/
-   npm start/ expo start
